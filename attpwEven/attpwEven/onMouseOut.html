onMouseOut

O evento onMouseOut executa um JavaScript quando o ponteiro do mouse é movido para fora de um elemento ou de seus filhos

Foi utilizado um botao com o id "cor" que quando o mouse sai de cima dele chama uma função para mudar a cor do texto do botao.
